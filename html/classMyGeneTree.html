bool MyGeneTree::rootTree ( MyGeneNode *  rootSon = NULL)

Check if the tree is unrooted.

If the tree is unrooted, (root has three sons), add a root son and make it the parent of the second two original root sons to create a rooted tree.

If a node is given, use that as a root son.

Otherwise, return false.

Returns
True if tree is unrooted.
