bool MySpeciesTree::computeSubdivision ( vector< pair< int, int > >  dateMap,
bool  bootstrapOrdering,
bool  ultrametric,
vector< int > &  changedTimeSlices,
string &  errStr 
)

Compute the subdivision, assigning time slices and adding nodes to give all branches the same height.

All leaves are assigned to time slice zero. Internal nodes are sorted by distance and consecutively assigned time slices, with the root at the maximal time slice. If bootstrapOrdering is true, the bootstrap distances are used directly. If not, the branch length from getDistanceToFather function is used to calculate the distance to the leaves.

If a dateMap is given, the internal nodes indicated by breadthFirstOrdering have their time slices swapped.

Returns
false if the date map has errors
Parameters
dateMappairs of time slices to swap
bootstrapOrderingOrdering given from bootstrap values.
ultrametricreport error if not ultrametric
changedTimeSlicesassigned time slices swapped
errStrdescription of error, if any
